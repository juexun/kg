# 知识图谱项目计划书（草稿）

## 目标

项目的初衷是为了构建一个跨行业、跨学科的知识平台；基于知识平台，为企业或个人用户构建上层应用。

## 盈利模式

- 前期
  - 领域知识图谱构建
  - 基于知识图谱为企业定制上层应用
- 中期
  - 知识服务平台化
- 后期
  - 商业咨询、BI
